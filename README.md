# MYTH Official Website
## GitHub Pages Deployment Guide

This directory contains the source code for the official MYTH landing page.

### ðŸš€ Quick Deployment

1. **Create a Repository**: Create a new public repository on GitHub named `myth-tools.github.io` (or a name of your choice).
2. **Push Files**: Upload `index.html` and `style.css` to the `main` branch.
3. **Enable Pages**:
   - Go to **Settings** > **Pages**.
   - Select **Deploy from a branch**.
   - Choose `main` branch and `/root` folder.
   - Click **Save**.
4. **Live URL**: Your site will be live at `https://myth-tools.github.io`.
